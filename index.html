Protocol Reference

Most Important IoT Protocols

MQTT
Message Queuing Telemetry Transport
Transport: TCP/IP Model: Publish-Subscribe Pattern: Broker-based QoS: 0, 1, 2 Use: Smart sensors, telemetry Exam Pattern: 8M comparison + diagram
Critical
CoAP
Constrained Application Protocol
Transport: UDP Model: RESTful Request-Response Methods: GET, PUT, POST, DELETE Port: 5683 Use: Constrained devices, M2M Exam Pattern: MQTT vs CoAP comparison
Critical
Zigbee
IEEE 802.15.4 based protocol
Standard: IEEE 802.15.4 Frequency: 2.4GHz / 868/915 MHz Topology: Star, Tree, Mesh Range: 10–100m Use: Smart home, automation Exam Pattern: Comparison with BLE, Wi-Fi
Very High
RFID
Radio Frequency Identification
Types: Active / Passive / Semi-passive Bands: LF (125kHz), HF (13.56MHz), UHF Components: Tag, Reader, Antenna Related: NFC Use: Supply chain, access control Exam Pattern: Working diagram + types
Repeated
6LoWPAN
IPv6 over Low-Power WPANs
Base: IEEE 802.15.4 Purpose: IPv6 on constrained devices Feature: Header compression MTU: 127 bytes adapted Use: Smart metering, WSN Exam Pattern: Definition + purpose 1M
Important
BLE
Bluetooth Low Energy
Version: BLE 5.0+ Frequency: 2.4GHz ISM band Power: Ultra-low energy Range: Up to 400m (BLE 5) Use: Wearables, health monitors Exam Pattern: Zigbee vs BLE comparison
Repeated
